## Fonctionnalités

### 🗂️ Gestionnaire de fichiers
- Scan complet du stockage Android / Windows
- Filtres par type : Images, Vidéos, Audios, APK, Documents, Gros fichiers
- Recherche en temps réel + tri par taille / nom / date
- Sélection multiple → suppression groupée avec confirmation
- Affichage de l'espace libéré

### 📥 Téléchargeur vidéo
- Coller un lien (YouTube, TikTok, Facebook, Instagram, Twitter...)
- Choix de qualité : 1080p / 720p / 480p / 360p
- Mode MP3 audio uniquement
- Suivi de progression en temps réel
